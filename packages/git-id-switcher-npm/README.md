# git-id-switcher

This is a namespace placeholder package.

The actual **git-id-switcher** is a VS Code extension that lets you manage and switch between multiple Git identities (name, email, SSH key, GPG key) per repository or workspace.

## Install the VS Code Extension

- [VS Code Marketplace](https://marketplace.visualstudio.com/items?itemName=nullvariant.git-id-switcher)
