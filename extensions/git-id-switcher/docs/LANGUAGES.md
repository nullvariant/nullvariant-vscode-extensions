This extension values linguistic diversity. We support not only major languages but also minority languages and even joke languages.

> **I value the existence of minorities.**
> I don't want to discard them just because they are small in number.
> Even if translations aren't perfect, I hope you can feel our intent to understand and show respect for minority languages.

## Minority Languages üåè

> **Why only three languages?**
>
> There are approximately 7,000 languages in the world, and nearly half of them face the risk of extinction within a few generations.
> I cannot translate into all of them.
>
> So I started with languages I have a personal connection to‚Äîplaces I've visited, cultures I've touched.
> Starting from small, personal connections is, I believe, the first step toward understanding minorities.
>
> - üå∫ **Hawaiian**: I was guided through sacred sites on Maui by a Native Hawaiian. At Honolulu Airport, I learned that Hawaiian is announced before English‚Äîa sign of respect for the local people.
> - üêª **Ainu**: I recently visited Hokkaido and saw the Ainu exhibitions at the Former Government Office.
> - üêâ **Ryukyuan**: I visit Okinawa frequently as a tourist.
>
> If you have a connection to an endangered language, please consider contributing a translation.
> Your small connection matters too.
>
> ‚ö†Ô∏è These translations are experimental and may contain errors.
> Native speakers are warmly invited to submit corrections via PR.

## Contributing Translations

We welcome translation contributions! See [CONTRIBUTING.md](./CONTRIBUTING.md) for guidelines.

### Priority Languages

We especially welcome contributions for:
- **Minority languages**: Help preserve linguistic diversity
- **Your native language**: Native speakers can significantly improve translation quality

### How to Contribute

1. Fork the repository
2. Create a new directory under `docs/i18n/{language-code}/`
3. Translate `README.md`
4. Submit a Pull Request
